# 🎨 Art Space App — Jetpack Compose

An interactive Android application built using **Jetpack Compose** that displays a curated collection of artwork images.
The app allows users to navigate through different artworks using intuitive controls while supporting multiple screen orientations.

> ⚠️ **Note:** This project was developed as part of my **learning phase** while exploring Android development with Jetpack Compose.

🔗 **Project Repository:**
https://github.com/chinmayijagadish/ArtSpace

---

## 📱 Features

* 🖼️ Display artwork images in a card layout
* 📝 Artwork title, artist name, and year information
* ⬅️➡️ Next and Previous navigation buttons
* 🔄 State-driven UI updates using Jetpack Compose
* 📐 Responsive design for **portrait and landscape** orientations
* 🎨 Modern Material Design UI components

---

## 🚀 Technologies Used

* **Kotlin**
* **Jetpack Compose**
* **Material 3**
* Android Studio
* State Management (`remember`, `mutableStateOf`)
* Responsive Layout using `LocalConfiguration`

---

## 🧠 Key Concepts Implemented

* Initial Composition & Recomposition
* State Management in Compose
* State Hoisting Pattern
* Layout Adaptation for Different Screen Sizes
* Component Reusability
